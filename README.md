## Installation

### Maven

```xml
<dependency>
  <groupId>ink.literate</groupId>
  <artifactId>turbawself</artifactId>
  <version>1.0.0</version>
</dependency>
```

### Gradle (Kotlin)

```kotlin
implementation("ink.literate:turbawself:1.0.0")
```

### Gradle

```groovy
implementation 'ink.literate:turbawself:1.0.0'
```
